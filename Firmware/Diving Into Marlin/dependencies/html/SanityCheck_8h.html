Test configuration values for errors at compile-time. Due to the high number of issues related with old versions of Arduino IDE we now prevent Marlin from compiling with older toolkits. We try our best to include sanity checks for all the changes configuration directives because people have a tendency to use outdated config files with the bleding edge source code, but sometimes this is not enough. This check will force a minimum config file revision, otherwise Marlin will not build.
