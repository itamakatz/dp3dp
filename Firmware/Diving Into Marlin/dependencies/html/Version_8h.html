Marlin release version identifier

#define SOURCE_CODE_URL   "https://github.com/MarlinFirmware/Marlin"

The SOURCE_CODE_URL is the location where users will find the Marlin Source Code which is installed on the device. In most cases —unless the manufacturer has a distinct Github fork— the Source Code URL should just be the main Marlin repository.

#define STRING_DISTRIBUTION_DATE   "2016-12-06 12:00"

The STRING_DISTRIBUTION_DATE represents when the binary file was built, here we define this default string as the date where the latest release version was tagged.

#define WEBSITE_URL   "http://marlinfw.org"

The WEBSITE_URL is the location where users can get more information such as documentation about a specific Marlin release.
